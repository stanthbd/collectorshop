# Expertise Équipe & Plan de Formation

## Recensement des compétences actuelles
- **Développement Web** : Expertise React, Node.js, TypeScript.
- **Backend** : Maîtrise des API REST, ORM (Prisma), PostgreSQL.
- **DevOps (Initial)** : Connaissance théorique de Docker et Kubernetes.

## Expertises techniques à acquérir
Le passage à une architecture micro-services à l'échelle nécessite d'approfondir :
1. **Kubernetes Intermédiaire** : Gestion des secrets (SealedSecrets), gestion fine des ressources (HPA), NetworkPolicies.
2. **Infrastructure as Code** : Maîtrise avancée de Terraform (State management via S3, Terragrunt).
3. **SRE (Site Reliability Engineering)** : Gestion des SLIs/SLOs via Prometheus.

## Plan de Formation Proposé
**Action de formation : "SecDevOps & Cloud Native Architecture"**
- **Durée** : 3 jours.
- **Formateur** : Expert Cloud.
- **Contenu** :
  - Jour 1 : Sécurisation du pipeline (SAST/DAST/SCA).
  - Jour 2 : Patterns de résilience (Circuit Breaker, Exponential Backoff).
  - Jour 3 : FinOps - Optimisation des coûts sur AWS EKS via Terraform.
- **Résultat attendu** : Autonomie de l'équipe sur la maintenance du cluster de production.
